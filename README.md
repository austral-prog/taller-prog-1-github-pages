# Taller de Programación 1 — GitHub Pages

Este repositorio es para que cada estudiante registre el link a su sitio personal de GitHub Pages.

## Instrucciones

### 1. Clonar el repositorio

```bash
git clone https://github.com/austral-prog/taller-prog-1-github-pages.git
cd taller-prog-1-github-pages
```

### 2. Editar este README

Agregá tu nombre y el link a tu GitHub Pages personal en la tabla de abajo, siguiendo el formato existente.

### 3. Guardar y subir los cambios

```bash
git add README.md
git commit -m "Add <tu nombre> GitHub Pages link"
git push origin main
```
